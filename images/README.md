# Estructura de carpetas de imágenes

Esta carpeta contiene todas las imágenes utilizadas en el proyecto IPStream. La estructura está organizada de la siguiente manera:

## /logos
Contiene los logotipos de la empresa y otros logotipos relacionados con el proyecto.
- `logo.png`: Logo principal de IPStream

## /banners
Contiene imágenes de banner para cabeceras y secciones destacadas.
- Banners promocionales
- Banners de secciones

## /icons
Contiene iconos utilizados en la interfaz de usuario.
- Iconos de características
- Iconos de servicios
- Otros iconos de UI

## /backgrounds
Contiene imágenes de fondo utilizadas en diferentes secciones del sitio.
- Fondos para secciones hero
- Texturas y patrones
- Fondos decorativos

## Convenciones de nomenclatura
- Utilizar nombres descriptivos en minúsculas
- Separar palabras con guiones (ej. `hero-background.jpg`)
- Incluir dimensiones en el nombre cuando sea relevante (ej. `logo-small.png`)
- Utilizar formatos optimizados para web:
  - `.webp` para imágenes con mejor compresión
  - `.png` para imágenes con transparencia
  - `.jpg` para fotografías
  - `.svg` para gráficos vectoriales

## Optimización
Todas las imágenes deben estar optimizadas para web antes de ser añadidas al repositorio.
